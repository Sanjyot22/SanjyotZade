LabelImg-TRK

"LabelImg-TRK" is an automated annotation tool, with deep learning based object tracking. LabelImg was an popular image annotation tool for object detection data annotation, however object detection and recognition data annotation can be a tedious task. This project aims to automate the annotation process by tracking the bounding box created in earlier image to the next consecutive image.

The process of data annotation is similar to labelImg. Now if tracking is turned on, the user has to create one bounding box across an image and the bounding on later images will automatically generated. Tracking automation only works on consecutive image of an video. If the consecutive images have different dimension, tracking will not be initiated. The tool has various tracking algorithms which can be selected from an dropdown menu. Some deep learning based trackers have also been integrated.

part-of-imagenet

This repository an open source project that can help you download portions of imagenet datasets within minutes, for your personal/profession custom object localization development. Most of the process is automated, all you need to do is run a few scripts, wait for download to complete and train your custom model.

Install requirements, download pre-requisites, tag the classes data to be downloaded & initiate the download process. At the end of download stats about every download will be provided. Downloads are all done simultaneously in batches(batch size is based on internet speed).
